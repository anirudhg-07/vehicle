## Overview
A simple Java Swing-based Parking Lot Management System that uses a Doubly Linked List to manage cars. It allows users to park, remove, and display cars in both forward and backward directions.

---

### Features
- Park and remove cars  
- View parked cars (Front → End and End → Front)  
- Dynamic parking lot capacity  
- Simple, clean GUI using Swing  

---

### Files
- `vehicle.java` — backend logic using doubly linked list  
- `ParkingLotGUI.java` — GUI interface built with Java Swing  

---

### Technologies Used
- Java  
- Java Swing (GUI)  
- Doubly Linked List (Data Structure)
